//告知AI的生成提示词
1.读取文档，生成相应的javascript数组
2.该数组将被存储到mongodb中，schema是
{
    title: String,
    choice: [String],
    rightIndex: Number,
    explanation: String,
    img: String
}
3.其中的rightIndex取值0~3，0表示A选项，3表示D选项
4.提取的数组要按照文档中的题目顺序
5.一个doc示例
{
  "title": "操作系统是对( )进行管理的软件。",
  "choice": [
    "软件",
    "硬件",
    "计算机资源",
    "应用程序"
  ],
  "rightIndex": 2,
  "explanation": "操作系统管理计算机的硬件和软件资源，这些资源统称为计算机资源。注意，操作系统不仅管理处理机、存储器等硬件资源，还管理文件，文件不属于硬件资源，但属于计算机资源。"
}
